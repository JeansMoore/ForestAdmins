Qu'est ce que OpenLayers ?
==========================

OpenLayers est une bibliothèque purement JavaScript pour afficher des cartes 
dans la plupart des navigateurs web, avec aucunes dépendances côté serveur.
OpenLayers implémente une `API JavaScript`_ pour construire des applications
géographique pour le web, similaire aux API de Google Maps et Virtual Earth de 
MSN, avec une différence d'importance -- OpenLayers est un logiciel Libre, 
développé pour et par la communauté des logiciels Open source.

Examples d'OpenLayers
---------------------

.. _`Examples d'OpenLayers` : http://openlayers.org/dev/examples/       

Un des principes qui guide le développement d'OpenLayers est de maintenir un 
ensemble d'exemple succint de la plupart des fonctionnalités, permettant une 
démonstration de ce que l'on peut réaliser avec la bibliothèque. Cela signifie 
que nos exemples sont typiquement la source de la documentation la plus à jour 
et fournie plus d'une 100e de portions de code à utiliser dans votre 
application.

Ces exemples sont disponibles à partir de la page `Examples d'OpenLayers`_.
